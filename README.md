~> About

This is a console-based C++ project I built for my Intermediate Programming class.
It simulates a simple hospital system where you can view/manage patients and physicians and update which doctor a patient is assigned to.
The program loads data from CSV files (edited in Microsoft Excel) and writes updates back to the patient CSV.

~>  What the Program Does
- Loads patient data from a CSV file
- Loads doctor (physician) data from a CSV file
- Maintains lists of patients and doctors using vectors
- Allows assigning or updating a doctor for a patient
- Displays patient and doctor information
- Uses a menu-driven console interface for user interaction
- Writes updated patient data back to the CSV file

~>  Data Files (CSV / Excel)
The program uses two CSV files that can be edited using Microsoft Excel.

~>  Expected file formats

Physician.csv

Patient.csv

These files should be placed in the same folder as the program (or inside a data folder if using relative paths).
